# ribbit

To start:

```bash
$ yarn install
```

To develop:

```bash
$ yarn run dev
```

To build for production:

```bash
$ yarn run build
```

To lint you code:

```bash
$ yarn run lint
```

## Motivation

I launched this proof-of-concept project to evaluate following projects:

* Vue
* Vuex
* Typescript bindings to Vue
* JEST testing framework 
* Websocket with RxJS extensions
